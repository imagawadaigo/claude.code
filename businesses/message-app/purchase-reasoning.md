# 購入理由の言語化と購入障壁

---

## 指名買いされる理由（一言）

> **「遠隔に住む家族と一緒に、相手への印象と感謝を、感動体験として届けられる唯一の手段」**

---

## ターゲット像

- 社会人2〜3年目、または親元を離れたばかりの新社会人
- 20代男女
- 「今年こそ何かしたい」と毎年思いながら結局プレゼントだけで終わっている人
- きょうだいがいて、バラバラに住んでいる

---

## 購入理由の3フレームワーク

### 1. 機能的メリット

- 遠隔に住む家族のメッセージをウェブ上で集められる
- 送料ゼロ・アプリDL不要・スマホだけで完結
- 複数人の写真＋メッセージが1つのページにまとまる
- 届けた後も永続URLで何度でも見返せる

### 2. 心理的変化

**before：**
家族という関係は長い時間をかけて形成されてきたが、離れて住むことで形骸化しつつある。  
「何かしてあげたい」気持ちはあるのに、面倒くさいが勝って毎年プレゼントだけで終わる。

**after：**
- 「そう思ってくれていたんだ」という再発見が家族の中で起きる
- 花をきっかけに「あの花何色だったっけ？」という会話が生まれる
- 形骸化していた家族チームに、感情と光が差し込まれる
- 「ちゃんと届けられた」という達成感と安堵が送った側に残る

> タックマンモデルで言えば、家族はすでにフォーミング〜ノーミング期を終えている。  
> このプロダクトは、形骸化したチームを機能期に引き戻す「感情的なリブート」をする。

### 3. 他との違いと共感ポイント

| 比較 | 他社（Memoreeel等） | このサービス |
|---|---|---|
| 伝えられること | テキスト・写真 | テキスト・写真＋**あなたへの印象** |
| 体験の質 | 静的なページを見る | 花が咲く感動体験として受け取る |
| パーソナライズ | なし | 花言葉クイズで相手のイメージを可視化 |
| 価格 | ¥935〜 | ¥400〜（人数ベース） |

**他社では足りない理由：**  
他社はメッセージを「送る」だけ。このサービスは「あなたのことをこう思っていた」という印象ごと届けられる。これは他に存在しない。

## 購入を検討している人が引っかかりそうな問い

### 使う前の不安

**Q1. 兄弟が本当に参加してくれるか不安**  
→ 招待してもスルーされたら、自分一人だけのページになってしまわないか？

**Q2. 払う前に演出を確認できないのか**  
→ 花が咲くと言われても、実際どんな感じかわからないまま払えない

**Q3. 使い方が難しくないか**  
→ 親や兄弟がITに詳しくなくても大丈夫か？

**Q4. 写真や個人情報は安全に管理されるか**  
→ 家族の顔写真をアップロードするのが不安

### 使う際の不安

**Q5. 投稿した後に修正・削除できるか**  
→ 誤字や写真を間違えたとき対応できるか

**Q6. 兄弟が投稿してくれなかったらどうなるか**  
→ 自分だけのページで母に送るのは恥ずかしい

**Q7. 決済した後にURLが届かなかったら**  
→ 払ったのに何も届かないトラブルへの不安

### 価値への疑問

**Q8. ¥○○払う価値があるのか**  
→ LINEでメッセージを送ればタダでできるのに、なぜお金を払う必要があるか

**Q9. 毎年使えるのか、それとも1回限りか**  
→ 父の日・誕生日でも同じように使えるか

**Q10. 受け取った母・父はどう思うか**  
→ 高齢の親がURLを開けるか、スマホ操作が心配

---

## 各引っかかりへの答え方（設計方針）

| 引っかかり | 解決手段 |
|---|---|
| Q1 兄弟が参加しない不安 | 「今〇人が投稿済み」の進捗をリアルタイム表示。1人でも送れることを明示 |
| Q2 演出を見たい | トップページにデモGIF/動画を必ず置く |
| Q3 使い方の難しさ | 招待URLを開くだけで投稿できる（ログイン不要）。手順を3ステップで説明 |
| Q4 プライバシー不安 | 「外部に公開されない・URLを知る人だけ見られる」を明示 |
| Q5 修正したい | 決済前であれば投稿内容を差し替え可能にする |
| Q6 一人になってしまう | 「一人でも使える」バリエーションを用意（自分だけの特別なメッセージとして） |
| Q7 決済後のトラブル | 決済完了画面に即時URLを表示。幹事用URLに戻ればいつでも確認できる |
| Q8 価値への疑問 | 「プレゼントの添え物として」「¥500のメッセージカード感覚」と訴求 |
| Q9 繰り返し使えるか | 「父の日・誕生日でもお使いいただけます」をトップに記載 |
| Q10 高齢の親が開けるか | 「URLをタップするだけ」。シンプルな受け取り体験を動画デモで見せる |

## 感想・口コミ収集の設計

家族はクローズドコミュニティであり、自然なバイラルは起きにくい。  
そのため、積極的に感動を引き出す仕掛けを置く。

受け取りページの下部に：
> 「この体験、誰かに教えてあげませんか？あなたの感動が、同じ思いを持つ誰かの背中を押すかもしれません。」

- SNSシェアボタン（X・Instagram）を置くが、**強制しない**
- アンケートの「感動リアクション」コメントを（許諾の上）ランディングページに掲載
