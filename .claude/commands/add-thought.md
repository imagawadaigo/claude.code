# 思考ログに追加

会話から生まれた思考・気づき・判断をNotionの思考ログに新しいページとして追加する。

## Notion構造

```
📁 今川大悟 — 思考・認知ベース（33b1ff54-005f-81cc-a194-c98ecfbf67fd）
└── 🗂️ 思考ログ（33b1ff54-005f-81f1-ac07-fc40440c5d13）
    ├── 📅 2026年4月（33b1ff54-005f-816b-bcf2-e01d13f975a1）
    └── 📅 [翌月以降は新規作成]
```

## 手順

1. **今日の日付を確認する**（YYYY-MM-DD形式）

2. **今月のフォルダが存在するか確認する**
   - 存在する → そのフォルダIDを使う
   - 存在しない → `notion-create-pages` で新しい月フォルダを「思考ログ」（`33b1ff54-005f-81f1-ac07-fc40440c5d13`）の下に作成

3. **思考ログページを作成する**
   - 親：今月のフォルダ
   - タイトル：`思考ログ | YYYY-MM-DD`（複数テーマあれば `思考ログ | YYYY-MM-DD | テーマ名`）
   - 内容：会話から抽出した思考・気づき・判断をまとめる

4. **ページの内容構成（目安）**
   ```
   ## 背景・文脈
   （何をきっかけにこの思考が生まれたか）

   ## 気づき・洞察
   （具体的な内容）

   ## 自分への問い / 次のアクション
   （あれば）
   ```

5. **完了報告**：作成したページタイトルとURLを伝える

## 注意

- 判断・意思決定の場合は `decisions/log.md` にも追記する
- 認知プロファイルや哲学の更新が必要と判断した場合は `/reflect-profile` または `/reflect-vision` も実行を提案する
